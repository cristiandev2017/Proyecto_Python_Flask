Sobre el Proyecto

Este proyecto tiene como objetivo realizar por medio de un lenguaje de programación calculos complejos. Para la realización del mismo se utilizaron 2 metodos númericos el metodo de newton y el metodo de bisección; Se utilizo el lenguaje Python como lenguaje base y utilizamos Flask para las interfaces graficas.
